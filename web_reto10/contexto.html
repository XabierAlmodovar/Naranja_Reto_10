Contexto del Reto

La empresa Lookiero se encuentra ante la necesidad de mejorar tanto la calidad de sus recomendaciones personalizadas como la experiencia digital que ofrece a sus clientas. Para lograrlo, es importante entender a fondo cómo interactúan las usuarias con la web, analizando los datos disponibles para detectar patrones de uso, dónde encuentran obstáculos y oportunidades de mejora en la experiencia de usuario.

Al mismo tiempo, la empresa quiere automatizar la generación de looks completos a partir de su catálogo. Esto implica crear un sistema que pueda combinar prendas siguiendo reglas claras y coherentes, utilizando teoría de grafos para entender cómo se relacionan los productos entre sí y qué combinaciones tienen sentido. Un desafío adicional es predecir cómo encajará una prenda nueva dentro de este ecosistema: si combina bien con otras, si tiene potencial para aparecer en looks y qué papel puede jugar dentro del catálogo.

A la vez, se busca caracterizar tanto a las clientas como a las prendas para anticipar la reacción de cada usuaria ante nuevos ítems, lo que implica analizar comportamientos, preferencias y atributos relevantes del producto. Todo ello responde a una problemática de conseguir mejorar la experiencia digital, automatizar de manera inteligente las recomendaciones y entender de verdad cómo interactúan las clientas con el producto. El objetivo final es ofrecer una experiencia más personalizada, eficiente y ajustada a lo que cada usuaria espera.


Objetivos Clave del Reto Lookiero

  • Análisis de la Experiencia de Usuario (UX)

    Estudiar el comportamiento de las usuarias para identificar puntos de fricción y patrones de uso relevantes. Se elaborarán informes y se propondrán mejoras concretas de navegación y experiencia.

  • Generación de Looks (Teoría de Grafos)

    Implementar un sistema basado en teoría de grafos y reglas de relación entre prendas para generar al menos 10 looks coherentes y de calidad.

  • Predicción de Relaciones entre Prendas

    Desarrollar un método que permita predecir las compatibilidades y conexiones de una prenda nueva con las ya existentes. Esta predicción es clave para mejorar las recomendaciones.

  • Caracterización de Datos de Look&Like (Opcional)

    Construir un modelo predictivo capaz de estimar la reacción (positiva/negativa) de la clienta hacia una prenda, usando la caracterización de clientas y productos para un sistema de recomendación más personalizado.
